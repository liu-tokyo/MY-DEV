## Thinkpad 13 20J1-CT01WW

型号 **20J1-CT01WW** 对应的正是 **ThinkPad 13 Gen 2**（第二代）。这是一款在 2017 年前后非常受欢迎的轻薄商务本。

### 核心硬件规格

| **组件**         | **详细参数**                         | **备注**                                |
| ---------------- | ------------------------------------ | --------------------------------------- |
| **处理器 (CPU)** | **Intel Core i3-7100U** (2核/4线程)  | 第7代 Kaby Lake 架构，基频 2.4GHz       |
| **显卡 (GPU)**   | Intel HD Graphics 620                | 集成显卡，支持 4K 视频解码              |
| **内存 (RAM)**   | **DDR4 2133MHz** (双插槽)            | 最大支持 **32GB** (16GB x 2)            |
| **硬盘 (SSD)**   | M.2 SATA 或 NVMe SSD                 | 默认通常为 128GB/256GB                  |
| **屏幕 (LCD)**   | 13.3英寸 (1366x768 或 1920x1080)     | **CTO1WW** 多为定制版，需确认是否为 IPS |
| **接口**         | 1x USB-C, 3x USB 3.0, HDMI, SD读卡器 | 接口非常丰富，支持 Type-C 充电          |
| **重量**         | 约 1.44 kg                           |                                         |

------

#### 这台机器的“优缺点”分析

#### **✅ 优点：**

1. **极强的扩展性：** 现在的 13 英寸笔记本内存大多是焊死的，但这台机器有**两个内存插槽**。如果你觉得卡，加一条 8GB 内存组成 16GB 双通道，体验会瞬间起飞。
2. **优秀的键盘手感：** 继承了 ThinkPad 经典的 1.8mm 键程，打字体验远好于你之前问的 Fujitsu U939。
3. **充电灵活：** 支持传统的圆口充电，也支持 **USB-C (PD 协议) 充电**，可以用现在的手机快充头（45W以上）供电。

#### **❌ 缺点：**

1. **CPU 性能瓶颈：** i3-7100U 只有 2 个核心。在 2026 年，如果同时开启浏览器、钉钉/微信和 Word，CPU 占用率会经常飙升到 100%。
2. **Win11 门槛：** 官方不支持 Windows 11。虽然能强行安装，但由于缺乏一些底层的安全指令集支持，流畅度不如 Windows 10。
3. **做工材质：** ThinkPad 13 系列属于入门级，机身主要是工程塑料（高配有金属顶盖），质感不如 T 系列或 X1 系列。

------

### 升级建议 (针对 2026 年使用)

如果你打算继续使用或购入这台机器，建议按以下顺序优化：

1. **内存升至 16GB：** 这是性价比最高的升级，能极大缓解 i3 处理器的压力。
2. **更换 NVMe SSD：** 如果原机自带的是 SATA 协议的 M.2 硬盘，换一个 NVMe 协议的（虽然接口可能是 PCIe 3.0 x2 速度），系统响应会更灵敏。
3. **系统选择：** 强烈建议留在 **Windows 10 LTSC** 或者使用轻量化的 **Linux (如 Ubuntu/Mint)**，这能让这颗 i3 处理器跑得更轻松。

#### 隐藏的第二个硬盘位 (WWAN 插槽扩展)

这台机器内部还有一个用于插上网卡（WWAN，即插流量卡上网）的空余插槽。

- **可能性：** 如果你不需要用 4G 上网功能，这个插槽**理论上**可以识别第二块 SSD。
- **限制条件：**
  - **尺寸：** 必须是 **M.2 2242**（短卡），常见的 2280 装不下。
  - **协议：** 这个插槽通常只支持 **M.2 SATA** 协议的 SSD（部分批次可能支持 NVMe，但极少，建议稳妥选 SATA）。
  - **兼容性：** 虽然很多 ThinkPad 支持这么干，但偶尔会有 BIOS 识别不到的情况。
