## 设备介绍

**Panasonic Toughbook CF-SX3** 是一款非常经典的“商务神机”（JAL日本航空等日企员工当年的标配）。虽然它发布于 2013-2014 年左右，但凭借极其扎实的做工、全面的接口和轻量化设计，直到今天在二手市场和复古折腾圈里依然很有热度。

以下是关于 CF-SX3 的核心亮点、配置以及折腾指南：

### 核心亮点

- **轻量与坚固兼得：** 采用了松下标志性的镁铝合金“褶皱”顶盖，能承受 100kgf 的加压和 76cm 的跌落。整机重量只有 **1.2kg - 1.4kg 左右**（取决于电池型号），这在当年带光驱的笔记本里是不可思议的。
- **变态级的接口拓展性：** 尽管机身轻薄，但它自带 **标准 HDMI、VGA、RJ45千兆网口、SD卡槽、USB 3.0 ×2、USB 2.0 ×1**，甚至还内置了一个**圆形弹出式 DVD 刻录光驱**。
- **超强续航与可换电池：** 支持热插拔电池，当年搭配大容量 L 电池的官方宣称续航能达到 20 小时以上（放到今天老电池退化，也能撑几个小时）。
- **极佳的键盘手感：** 标志性的“叶片”形状键帽，键程极深，打字非常舒服。

### 硬件配置参数

- **处理器：** 第 4 代 Intel Core i5-4200U / i5-4300U 或 i7-4500U（双核四线程，Haswell 架构）。
- **内存：** 板载 4GB（部分高配板载 8GB），同时**带有一个标准的 DDR3L 内存插槽**，最高可以再加一根 8GB，组成 12GB 或 16GB 内存。
- **屏幕：** 12.1 英寸 16:9 雾面屏，分辨率为 **1600 × 900**。虽然不是 IPS 广视角，但防眩光效果在当年很实用。
- **硬盘：** 原装通常是 2.5 寸机械硬盘或早期 SSD，接口为标准 **SATA 3.0**，非常容易拆解升级。

### 这台机器今天怎么“折腾”？

如果您手里有一台 CF-SX3，它非常适合用来做以下几件事：

#### 1. 硬件升级（让它流畅续航）

- **换装 2.5 寸 SATA SSD：** 这是最立竿见影的升级。CF-SX3 换硬盘非常简单，拔掉电池，解开底部的挡板滑块，直接就能把旧硬盘抽出来换上新的 SSD。
- **加满内存：** 建议在空闲插槽上插一根 DDR3L 1600MHz 8GB 内存，日常轻办公、查资料绝对不会卡顿。

#### 2. 系统选择：极其适合 Linux (如 Ubuntu)

虽然它自带的是 Windows 7 或 Windows 8 标签，但由于硬件全面使用 Intel 芯片组（网卡、显卡、处理器），**它对 Linux 系统的兼容性近乎完美**。

- 刷入 **Ubuntu 22.04 LTS 或 24.04 LTS**，作为一台轻量级的便携代码机、移动服务器、或者专门用来跑 Git 仓库和文件管理的机器，效率极高。
- Haswell 的核显（Intel HD 4400）在 Linux 下硬解 1080P 视频毫无压力，日常看文档、刷网页非常省电、安静。

#### 3. 经典备用机 / 移动工具箱

因为自带 VGA、HDMI 和网口，在需要调试老旧设备、机房连交换机、或者接投影仪演讲时，它不需要任何拓展坞（Dongle），是极其可靠的“瑞士军刀”式备用机。
