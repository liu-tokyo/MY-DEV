# FUJITSU LIFEBOOK U939/B

> 型号：`FMVU24013`  
> FMVU24013** 是富士通（Fujitsu）**LIFEBOOK U938/S** 系列中的一个具体型号代码。  
> 这款机器是轻薄本界的“奇葩”存在——它以**极致轻量化**著称。即便在 2026 年，它那不到 800 克的重量依然让许多现代轻薄本感到汗颜。



## 1. 配置信息

| 硬件         | 配置                                                         | 备注                                                         |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 处理器 (CPU) | Intel Celeron 4305U                                          | [**1309**/**1562**](https://www.cpubenchmark.net/cpu.php?cpu=Intel+Celeron+4305U+%40+2.20GHz)<br />专为极轻薄办公设计的低功耗芯片。 |
| 内存 (RAM)   | 8GB DDR4                                                     |                                                              |
| 硬盘 (SSD)   | 256GB M.2 SSD                                                | 通常为 SATA 协议，但接口支持 NVMe。                          |
| 显卡 (GPU)   | Intel UHD Graphics 610                                       |                                                              |
| 屏幕         | 13.3 英寸 LCD 雾面屏                                         |                                                              |
| 分辨率       | 1920 x 1080 (FHD)                                            |                                                              |
| 重量         | 约 777g - 830g                                               | 这是它最大的价值所在。哪怕 CPU 再弱，它依然是**世界上最轻的笔记本之一**。 |
| 接口与扩展   | **USB-C：** 2 个（通常支持 **PD 充电**和视频输出）。<br />**USB-A：** 2 个 (USB 3.1)。<br />**视频输出：** 标准尺寸 **HDMI**。<br />**网络：** 富士通标志性的**抽拉式 RJ45 网口**。<br />**生物识别：** 键盘右下角通常配有**指纹识别**或掌纹识别。 |                                                              |

※ **性能瓶颈：** 由于 **Celeron 4305U** 不支持睿频，它的瞬时响应速度较慢。
