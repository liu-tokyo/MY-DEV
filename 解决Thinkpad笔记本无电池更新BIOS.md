# 解决Thinkpad笔记本无电池更新BIOS

💡ThinkPad T480 的 BIOS 更新通常要求**电池电量充足（例如至少 30% 或更高）并连接交流电源适配器**，以防止更新过程中断电导致系统永久性损坏。

如果电池耗尽且无法充电或更新，您可以尝试以下方法：

## 🛠️ 强制绕过电池检查（高级用户谨慎操作）

对于某些 ThinkPad 型号和 BIOS 版本，可以通过命令行参数**强制**执行 BIOS 更新，从而绕过电池电量检查。这种操作**存在风险**，请务必确保在更新过程中**电源适配器连接稳定**。

1. **下载 BIOS 更新文件：**

   - 访问联想（Lenovo）官方支持网站，找到 ThinkPad T480 的最新 BIOS 更新程序（通常是 `.exe` 文件）。

2. **解压更新文件：**

   - 双击下载的 `.exe` 文件，程序会提示您解压到一个临时文件夹（例如 `C:\DRIVERS\FLASH\XXXX`）。**记下或复制**这个解压路径。

3. **打开命令提示符（管理员）：**

   - 在 Windows 搜索栏输入 **`cmd`**，右键点击“命令提示符”，选择**“以管理员身份运行”**。

4. **导航到解压路径：**

   - 在命令提示符中，输入 `cd <您记下的解压路径>`，例如：

     ```
     cd C:\DRIVERS\FLASH\XXXX
     ```

     （请将 `XXXX` 替换为实际的文件夹名称）

5. **执行强制更新命令：**

   - 运行以下命令来启动更新程序并添加绕过检查的参数：

     ```
     winuptp64 -sp
     ```

     - **`winuptp64`** 是实际的更新程序名称（名称可能因 BIOS 版本而异，但通常是这个）。
     - **`-sp`** 是绕过电池电量检查的参数。

6. **遵循屏幕提示：**

   - 更新程序启动后，请仔细阅读所有警告和提示，然后开始更新。**在更新完成前，绝对不要断开电源！**

> **⚠️ 风险警告：** 如果在强制更新过程中由于电源适配器松动或突然断电，**您的主板可能会变砖（无法启动）**，需要联系维修中心更换主板。

## 🔋 解决电池问题使其可以充电

如果电池只是暂时耗尽且无法充电（例如出现“已插入，未充电”），您可以尝试以下方法让其恢复充电，从而满足 BIOS 更新的正常要求：

1. **重置内置电池（通过 BIOS）：**
   - **关闭**电脑并连接交流电源适配器。
   - 开机时按 **Enter**（或 **F1**，具体取决于您的启动设置）进入 BIOS 设置界面。
   - 导航到 **Power** 菜单。
   - 找到 **Disable built-in battery**（禁用内置电池）选项并选择它。
   - 电脑会**自动关机**。
   - 等待 30 秒到 1 分钟后，重新插拔电源适配器，然后开机。这有时可以重置电池状态并恢复充电。
2. **检查电池状态和驱动：**
   - 如果 T480 **有两块电池**（内置和外置），请确保外置电池已正确安装。
   - 进入 **设备管理器**，展开 **电池** 类别，右键点击 **Microsoft ACPI 兼容的控制方法电池**，选择**“卸载设备”**（不要勾选“删除此设备的驱动程序软件”），然后**重启**电脑。Windows 会自动重新安装驱动。

如果尝试所有方法后电池仍然无法充电或保持电量，那么您可能需要**更换电池**才能安全地执行 BIOS 更新。
