# FUJITSU LIFEBOOK SH90/P

> 型号：FMVS90PWD1

## 1. 配置信息

这是一台发布于 2014 年左右的高端轻薄商务本（Ultrabook），主打超高分辨率屏幕和日本原装制造。

### 1.1 核心配置信息

  | 组件              | 详细参数                                                     |
  | ----------------- | ------------------------------------------------------------ |
  | 处理器 (CPU)      | Intel Core i5-4200U (第4代 Haswell, 1.6GHz, 睿频 2.6GHz)     |
  | 内存 (RAM)        | 标准 4GB (板载)，最大可扩展至  12GB (含一个插槽)             |
  | 硬盘 (Storage)    | 约 500GB HDD (5400转) 或 混合硬盘  (SHDD)                    |
  | 显示屏  (Display) | 13.3英寸 WQHD 触控屏，分辨率 2560 x 1440 (IGZO 技术)         |
  | 显卡 (GPU)        | Intel HD Graphics 4400 (集成显卡)                            |
  | 光驱              | 内置 DVD Super Multi 刻录机  (可拆卸换成减重模块或第二块电池) |
  | 操作系统          | 原装 Windows 8.1 64位 (现多被升级为 Win 10)                  |
  | 重量              | 约 1.59 kg                                                   |

### 1.2 亮点与槽点

- **屏幕非常出色：** 在那个年代就配上了 2K 级别的 IGZO 屏幕，画质细腻度和色彩表现即便放在今天也算不错。
- **多功能舱设计：** 富士通的特色，光驱位可以热插拔，换成散热风扇、减重器或者扩容电池。
- **接口极其丰富：** 虽然是 13 寸轻薄本，但自带标准网口 (RJ45)、VGA 和 HDMI，非常适合办公。
- **性能局限：** 由于使用的是 4 代 i5 低压版处理器和机械硬盘，在运行现在的 Windows 10/11 时会感到明显的卡顿。



## 2. 硬件升级

### 2.1 关键配件兼容型号

| 部件         | 兼容规格                       | 建议建议                                                     |
| ------------ | ------------------------------ | ------------------------------------------------------------ |
| 内存 (RAM)   | DDR3L 1600MHz (1.35V 低电压版) | 机器自带 4GB 板载，剩余 1 个插槽。最高可加装一根 8GB，组成 12GB。 |
| 硬盘 (SSD)   | 2.5 英寸 SATA3 接口 固态硬盘   | 原装是机械硬盘，换成 2.5 寸 SATA 固态（如三星 870EVO 或英睿达 MX500）速度提升最明显。 |
| 电池 (Main)  | 型号：FPB0307S / FPCBP405      | 77Wh 大容量电池，如果续航不行了可以搜这个型号。              |
| 多功能仓电池 | 型号：FPCBP416                 | 这是装在光驱位的第二块电池，选配配件。                       |

### 2.2 升级建议

如果你打算继续使用这台机器，**强烈建议**进行以下两项升级，会有“重获新生”的感觉：

1. **更换 SSD：** 将原装 HDD 换成 SATA 接口的固态硬盘（SSD）。
2. **增加内存：** 在空的内存插槽里加一根 8GB DDR3L 1600MHz 的内存条，凑成 12GB。

### 2.3 简易拆解步骤图示

富士通 SH90 系列的拆解逻辑分为“外挂模块”和“内部底壳”两部分。

#### 2.3.1：免工具拆卸（更换内存/清理风扇）

1. **断电：** 关机并取下主电池。
2. **内存盖：** 底壳正中央有一个小盖板，拧掉一颗螺丝即可看到内存插槽。
3. **风扇清理：** 富士通设计了一个非常人性化的“灰尘挡板”（位于出风口附近的小塑料片），拔掉它就可以直接清理散热鳍片的积灰，无需拆整机。
4. **光驱位：** 拨动侧面的锁扣，可以直接把 DVD 光驱拔出来，换成硬盘托架或第二块电池。

#### 2.3.2：深度拆卸（更换 SSD/涂抹硅脂）

1. **卸下底壳：** 拧下底面可见的所有螺丝。**注意：** 隐藏在脚垫或光驱位内部可能还有细小螺丝，切勿蛮力撬开。
2. **硬盘位置：** 拆开底壳后，你会看到 2.5 英寸硬盘位，通常有防震架固定，直接断开 SATA 排线即可更换。
3. **注意排线：** 富士通的内部排线非常细且脆，尤其是触控板和键盘的排线，翻开底壳时动作要轻。

------

#### 2.3.3. 特别注意事项

- **内存电压：** 一定要买 **DDR3L**（L 代表 Low Voltage 1.35V）。如果买成标准的 DDR3 (1.5V)，可能会导致无法开机或频繁蓝屏。
- **屏幕排线：** 这一代的 2K 屏幕排线接口比较特殊，如果拆机涉及主板移动，千万不要拉扯屏幕连接处，否则那个 IGZO 屏一旦损坏，维修成本极高。
- **螺丝长短：** 拆下的螺丝建议按位置摆放，这台机器不同位置的螺丝长度不一，装错可能会顶穿外壳。
