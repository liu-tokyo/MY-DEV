## 进入BIOS设置的办法

根据制造商，有所不同

- 「F2」键的制造商
  NEC、富士通、東芝、SONY、Lenovo(IBM)、Dell、日立、Panasonic、Mouse Computer、Sharp、ASUS、Intel、ASROCK 、Gateway、acer

- 「Del」键的制造商
  Epson、ASUS、Gigabyte、ESC、MSI、ASROCK(也有`F2`的情况)

- 「F10」键的制造商
  HP(Compaq)

- 「Ctrl+Alt+Enter」键的制造商
  Dell

- 【ENTER】键的制造商

  Lenovo的新系列（Thinkpad主要系列之外）
