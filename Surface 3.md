## Surface 3



### 1. 配置信息

**Surface 3** 是微软发布的一款 2合1平板电脑，适用于轻度办公和日常娱乐。它的配置相对于其他型号来说，定位较为入门级，适合需要轻便和一定性能的用户。以下是 **Surface 3** 的主要配置信息：

| 硬件项目           | 硬件信息                                                     | 备注                                                         |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 处理器 (CPU)       | Intel Atom x7-Z8700                                          | 四核，1.6GHz，最大加速频率 2.4GHz<br />适用于轻度的办公和浏览网页，不适合运行重型应用。 |
| 内存 (RAM)         | 4GB LPDDR3 RAM                                               | 适合基础任务，可能会在多任务处理时出现性能瓶颈。             |
| 存储               | 128GB eMMC<br />型号：Samsung MDGAGC                         | 相对于 SSD 而言，eMMC 的读写速度较慢，但对于轻量级任务和存储来说足够用。 |
| 显示屏             | **10.8 英寸** **PixelSense 屏幕**                            | 分辨率：**1920 x 1280 像素**，3:2 比例<br />显示效果较好，尤其适合浏览网页和文字处理。 |
| 图形处理单元 (GPU) | Intel HD Graphics 405                                        | 集成显卡，适合基本的图形任务，如视频播放、浏览网页等。       |
| 操作系统           | 初始安装为 **Windows 8.1**                                   | 支持升级至 **Windows 10** 和 **Windows 11**（不过性能可能受限于硬件） |
| 电池续航           | 大约 10 小时的本地视频播放                                   | 电池续航较为不错，适合日常使用。                             |
| 端口               | 1 个 **Micro USB 3.0** 端口（用于充电和数据传输）<br />1 个 **Full-size USB 3.0** 端口<br />1 个 **Mini DisplayPort**（用于外接显示器）<br />1 个 **MicroSD 卡插槽**<br />1 个 **3.5mm 耳机插孔**<br />**Surface Connect 接口**（用于连接外部设备，如 Surface Dock） |                                                              |
| 摄像头             | **前置 3.5MP 摄像头**（适合视频通话）<br />后置 8MP 摄像头   |                                                              |
| 重量和尺寸         | **重量**：约 **622 克**<br />**尺寸**：**267 x 175 x 8.7 mm** | 设计轻便，适合外出携带。                                     |
| Wi-Fi 和蓝牙       | Wi-Fi 802.11ac<br />蓝牙 4.0                                 |                                                              |
| 其他特点           | 支持 **Surface Pen**（需另购）<br />支持 **Windows Hello**（仅限前置摄像头） |                                                              |

**Surface 3** 在发布时定位为一款性价比相对较高的轻便设备，适合轻办公、娱乐、便携性需求较高的用户。
