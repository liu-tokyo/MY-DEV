# HW-01K (Huawei P20 Pro for docommo)

> 华为内部版本：`CLT-L0J`

#### 删除 `「dマーケット」`

- 指令

  ```
  pm uninstall -k --user 0 com.nttdocomo.android.store
  ```

  ※ 软件包名称：`com.nttdocomo.android.store`

  如果显示“Success”就OK了

#### 显示网络速度

- 指令

  ```
  settings put system show_network_speed_enabled 1
  ```
