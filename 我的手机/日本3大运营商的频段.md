# 日本 3 大运营商的频段

## NTT DoCoMo 周波数

### NTT DoCoMo - 5G

|                                                 | **周波数帯**   | **周波数**  | **備考**                      |                       |
| -------------- | ----------- | ------------------------------ | ----------------------------------------------- | ----------------------------------------------- |
| **Sub-6** | **n78**     | 3.6～3.7GHz                    | 国際的に主要となる周波数帯 n78はn77に内包される | 称为Sub-6，它是国际5G的主要频段。<br/>想要使用5G，支持n78的终端必不可少。 |
| **n79**        | 4.5～4.6GHz | ドコモだけ対応している周波数帯 |                                                 | 这是一个称为 Sub-6 的频段。<br/>在主要运营商中，只有 DoCoMo 兼容。<br/>docomo好像在开发n79上的5G，所以如果你想在docomo线上用5G，最好先看看是否支持n79。 |
| **ミリ波**     | **n257**    | 27.4～27.8GHz                  | 5Gの真価を発揮する周波数帯 対応エリアは希少     | 这是一个称为毫米波的频段。<br/>是实现“高速大容量”、“低时延”、“多路同时连接”所必需的频段，这些都是5G的终极目标。<br/>因为是高频，所以很容易受到障碍物的影响，所以一直到现在都没有使用，但是现在为了获得舒适的通讯环境而使用。<br/>由于容易受到障碍物的影响，因此很难在大范围内均匀支撑，因此将主要在市中心等主要地点进行支撑。<br/>由于作为基础技术的5G还没有普及，毫米波的普及还需要时间。 |

## Softbank 周波数

### Softbank - 5G

|           | 周波数          | 優先度 | 備考                         |                                                              |
| --------- | --------------- | ------ | ---------------------------- | ------------------------------------------------------------ |
| Band n77  | 4.0～4.1GHz帯   |        | ソフトバンクの5Gメインバンド | n77 频段，称为 Sub6，是使用 4G 基站部署的 5G 频段。<br/>由于它使用的是4G基站，所以有很多地方可以和5G通信的优势，但是没有毫米波快，后面会讲到。<br/>使用Softbank的5G通讯时，首先要确认你的智能手机是否支持n77。 |
| Band n257 | 29.1～29.5GHz帯 |        | ミリ波                       | 毫米波是展示 5G 真正价值所必需的频段。<br/>・高速大容量<br/>・低延迟<br/>・多个同时连接<br/>为了实现这一点，支持毫米波是必不可少的。<br/>因为容易受到障碍物的影响，所以比上面提到的n77可以使用的地方更有限（主要在市区扩展），但是支持毫米波会大大提高通信速度。 |

## Au(KDDI) 周波数

### Au(KDDI) - 5G

| 周波数帯（バンド） | 周波数             | 優先度 | 備考             |                                                              |
| ------------------ | ------------------ | ------ | ---------------- | ------------------------------------------------------------ |
| Band n77           | 4.0～4.1GHz        | 〇     | n78を含む        | 它是使用 4G 线路的称为 Sub6 的频段。<br/>虽然和n78相比不是主频段，但是兼容的话可以交流的更舒服。<br/>检查您的智能手机是否兼容。 |
| Band n78           | 3.4～3.7GHz帯      | ●      | auのメインバンド | au的5G线路主频段。<br/>如果您想在 au 线路上使用 SIM free 智能手机进行 5G 通信，请先检查它是否支持 n78。 |
| Band n257          | 27.8GHz～28.2GHz帯 | ●      | ミリ波           | 毫米波是全面5G通信必不可少的频段。<br/>●高速大容量<br/>●低延迟<br/>●多个同时连接<br/>为此，必须支持毫米波。 不过可以使用的区域基本都是市区，相对于Sub6来说是有限的。<br/>如果您选择同时支持Sub6和毫米波的智能手机，您可以毫无问题地使用au的5G通信。 |

### Au(KDDI) - 4G

| 周波数帯（バンド） | 特徴     | 優先度 | 備考                                   |                                                              |
| ------------------ | -------- | ------ | -------------------------------------- | ------------------------------------------------------------ |
| Band 1             | 2.1GHz帯 |        | auのメインバンド                       | 频段 1 是所有三个运营商支持的主要频率。<br/>在使用 au 线之前，请先检查它是否支持 Band 1。 |
| Band 3             | 1.7GHz帯 |        | 東京・名古屋・大阪メインで使われている | DoCoMo 和 Rakuten Mobile 也支持该频段。<br/>主要部署在东京、大阪、名古屋等大城市。<br/>海外生产的智能手机大多支持Band 3，购买时一定要确认。 |
| Band 11            | 1.5GHz   |        | 対応エリアが狭い                       | 在有限区域部署的频段。<br/>即使不对应，也不会有什么特别的问题。 |
| Band 18            | 800MHz   |        | auのメインプラチナバンド               | 它被称为白金带，如果不兼容，则在“易于连接”方面存在差异。<br/>特别是Band 18是au的主波段，使用au线时必须支持。<br/>如果它不支持 Band 18，则会出现服务中断或速度极慢等情况。 |

| Band 28            | 700MHz帯 |        | 地域限定のプラチナバンド               | 白金频段的优先级较低，因为它仅部署在较小的地理区域。         |
| Band 41            | 2.5GHz帯 |        | WiMAX 2+                               | 频段 41 是 WiMAX 的频率。<br/>在使用UQWiMAX等WiMAX服务时，支持它是必不可少的，但如果你不使用它，你也不用担心。 |
| Band 42            | 3.5GHz帯 |        | キャリアアグリゲーション時に使用される | 一种用于载波聚合的频率，可通过同时使用多个频率实现高速通信。<br/>不需要支持，但如果您想更快地沟通，请注意。 |
