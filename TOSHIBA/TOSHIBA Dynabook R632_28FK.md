## 2. 系统安装

### 2.1 UEFI 模式安装

在安装前，你必须先进入 BIOS 修改设置，否则安装程序可能无法以 UEFI 模式引导。

1. **进入 BIOS：** 关机状态下按住 **`F2`** 不放，再点电源键。
2. **切换模式：** 找到 **Advanced** -> **System Configuration** -> **Boot Mode**，将其改为 **UEFI Boot**。
3. **关闭安全启动：** 在 **Security** 选项卡中，将 **Secure Boot** 设置为 **Disabled**（早期 UEFI 对 Win10 的安全启动兼容性较差，先关掉能大幅提高成功率）。
4. **保存：** 按 **`F10`** 保存并退出。

- 使用 **优启通** 自带的 安装工具，无法进入系统；

  没必要
