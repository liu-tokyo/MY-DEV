

## 1. 查询磁盘固件

- `PowerShell` 以管理员身份运行，输入如下命令：

  ```powershell
  Get-WmiObject -Class Win32_DiskDrive | Select-Object Model, FirmwareRevision
  ```

  结果大致如下：

  ```
  Model                         FirmwareRevision
  -----                         ----------------
  TOSHIBA DT01ACA200 USB Device MX4O
  asmedia ASMT1153e USB Device  0
  CSSD-M2B1TPG3VNF              EGFM13.2
  INTEL SSDSC2BX016T4R          G201DL2E
  ```

※ 需要注意的是，CFD的SSD的固件更新，确实会删除所有数据。
