## Surface Pro 3

### 1. 配置信息

**Surface Pro 3** 是微软推出的一款高性能 2合1 平板电脑，专为商务和专业用户设计。  
相比 Surface 3，Surface Pro 3 更加注重性能和多任务处理能力，配备了更强的硬件，适合处理更复杂的工作负载。以下是 **Surface Pro 3** 的详细配置信息：

| 硬件项目           | 硬件信息                                                     | 备注                                                         |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 处理器 (CPU)       | Intel Core i5-4300U                                          | 双核，1.9GHz，最大 2.9GHz                                    |
| 内存 (RAM)         | 4GB LPDDR3 RAM                                               | 提供更强的多任务处理能力，适合办公、浏览、轻度视频编辑等任务。 |
| 存储               | 128GB SSD                                                    | 相较于 eMMC，SSD 提供了更快的读写速度，提升了整体性能。      |
| 显示屏             | 12 英寸 PixelSense 屏幕                                      | 分辨率：**2160 x 1440 像素**，3:2 比例<br />屏幕清晰，色彩准确，适合文档编辑、图片查看等任务。 |
| 图形处理单元 (GPU) | Intel HD Graphics 4400（i3 和 i5 版本）                      | 提供基础的图形处理能力，适合轻度的图形和视频编辑。           |
| 操作系统           | 初始安装为 **Windows 8.1 Pro**                               | 支持升级至 **Windows 10** 和 **Windows 11**。                |
| 电池续航           | 约 9 小时的视频播放                                          | 续航表现不错，适合全天使用。                                 |
| 端口               | **1 个 USB 3.0** 端口<br />**Mini DisplayPort**（可连接外部显示器）<br />**microSD 卡插槽**（支持高达 128GB 卡）<br />3.5mm 耳机插孔<br />**Surface Connect 接口**（用于连接 Surface Dock）<br />Cover键盘连接器 |                                                              |
| 摄像头             | **前置 5MP 摄像头**（适用于视频通话）<br />**后置 5MP 摄像头**（支持高清视频录制） |                                                              |
| 重量和尺寸         | **重量**：约 **800 克**（根据配置不同略有差异）<br />**尺寸**：**292.1 x 201.4 x 9.1 mm** | 较为轻便，但由于配备较大屏幕，整体便携性略低于 Surface 3。   |
| Wi-Fi 和蓝牙       | Wi-Fi 802.11ac<br />蓝牙 4.0                                 |                                                              |
| 其他特点           | 支持 **Surface Pen**（另售，支持倾斜、压感和无线充电）<br />支持 **Windows Hello**（仅限前置摄像头）<br />**立式支架**（可以将 Surface Pro 3 轻松调节为多个角度，适合打字和观看视频）<br />**支持触控屏和键盘附件**（Surface Type Cover） |                                                              |

#### 性能与使用场景：

- **Surface Pro 3** 作为一款 2 合 1 设备，兼具平板与笔记本功能，非常适合那些需要高性能和便携性的专业用户。
- 它在 **多任务处理**、**文档处理**、**媒体创作**、**编程**、**视频编辑**等领域表现优越，是一款能替代传统笔记本电脑的强大设备。

#### 总结：

Surface Pro 3 是一款非常强大的 2 合 1 设备，凭借 **高分辨率屏幕**、**强劲的处理器** 和 **充足的内存配置**，它适合在各种专业环境中使用。即使现在已经是几代之后的产品，依然能够满足大部分用户的日常需求。
